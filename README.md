# ReefGuide

## Quick start

We are using pnpm, so install it (using npm, ideally at v20 +)

```
npm install pnpm -g
```

## Install dependencies

```
pnpm i
```

## Run turbo commands

Using the installed turbo

```
npx turbo ...
```

Or global install 

```
npm turbo -g
turbo ...
```

## Build 

```
turbo build
```
